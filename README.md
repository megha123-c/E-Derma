
# 🩺 E-Derma – AI-Based Skin Analysis System

E-Derma is an AI-powered web application that performs real-time facial skin analysis and provides personalized skincare recommendations. Users can capture a live image using a webcam or upload an existing image for analysis. The system uses image processing techniques with a pre-trained ONNX model and rule-based analysis to detect common skin conditions such as acne, pigmentation, wrinkles, dryness, and different skin types. Based on the analysis, it recommends suitable skincare ingredients and helps users locate nearby dermatologists.

## ✨ Features
- 👤 User Registration & Login
- 📝 User Profile Management
- 📷 Real-time Webcam Capture
- 🖼️ Image Upload for Skin Analysis
- 🤖 AI-Based Skin Condition Detection
- 🧴 Personalized Skincare Recommendations
- 🛍️ Product Recommendations
- 📍 Nearby Dermatologist Finder
- 📊 Analysis History
- ⚙️ Admin Dashboard

## 🛠️ Tech Stack
- 🌐 Frontend: HTML, CSS, JavaScript
- ⚡ Backend: FastAPI (Python)
- 🖼️ Image Processing: OpenCV
- 🤖 AI Model: ONNX
- 🗄️ Database: SQLite
- 🗺️ API: Google Maps API

## 🚀 Workflow
1. Register or log in.
2. Capture or upload a facial image.
3. Preprocess the image using OpenCV.
4. Analyze the image using the ONNX model and rule-based logic.
5. Detect skin type and skin conditions.
6. Generate personalized skincare and product recommendations.
7. Display nearby dermatologists based on the user's location.
